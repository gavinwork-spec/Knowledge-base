# 手动触发图纸分类命令

## 命令概述
触发图纸自动分类脚本，对数据库中的图纸进行产品类别和标准件/定制件标识分类。

## 使用方法
```
/classify_drawings
```

## 参数选项
- `--reclassify`: 重新分类所有图纸（包括已分类的）
- `--stats`: 仅显示当前分类统计信息
- `--report`: 生成详细的分类报告

## 功能描述
1. **自动分类**: 根据图纸名称自动识别产品类别
2. **定制件标识**: 识别标准件和定制件
3. **置信度评估**: 计算分类结果的置信度
4. **批量处理**: 支持批量处理大量图纸
5. **结果导出**: 生成分类报告和统计信息

## 分类规则
基于以下关键词匹配：
- **紧固件**: 螺栓、螺钉、螺母、垫圈、销、铆钉
- **家具**: 椅子、桌子、沙发、柜子、床
- **建材**: 钢材、木材、瓷砖、涂料、防水保温
- **定制件**: 定制、异形、特殊、非标、来图

## 输出结果
- 分类统计：总数量、已分类数、分类率
- 类别分布：各类别的图纸数量
- 置信度分析：高中低置信度的分布
- 详细报告：保存到 `data/processed/` 目录

## 示例输出
```
✅ 图纸分类完成!
📊 处理总数: 408
🏷️ 成功分类: 39
❓ 未分类: 369
📈 分类率: 9.6%
📄 详细报告: data/processed/classification_report_20251105_172704.json
```

## 注意事项
- 分类基于文件名关键词匹配，准确率取决于文件命名规范
- 建议定期检查分类结果，修正错误分类
- 低置信度的分类记录需要人工审核

## 相关文件
- `classify_drawings.py`: 分类脚本
- `knowledge/classification_rules.md`: 分类规则文档
- `data/processed/`: 分类结果输出目录
