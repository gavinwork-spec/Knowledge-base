## 🤖 四大核心智能体

### 1️⃣ fastener_drawing_analysis_agent.yaml
**智能紧固件图纸分析专家**

**核心能力：**
- 🔍 PaddleOCR + FreeCAD 深度技术分析
- 📏 紧固件标准库 (ISO, DIN, ANSI, GB)
- 🏭 供应商能力匹配和推荐
- ⏱️ 15分钟内完成可行性评估

**技术特色：**
- 支持多种文件格式 (.pdf, .dwg, .dxf, .step)
- 自动识别螺栓类型、尺寸、强度等级
- 质量标准合规性检查
- 实时供应商匹配算法

### 2️⃣ raw_materials_price_agent.yaml
**原材料价格监控与预测专家**

**核心能力：**
- 📈 实时LME金属价格监控
- 💱 多汇率风险分析
- 🔮 Prophet时间序列预测
- 💰 智能报价优化

**覆盖材料：**
- 碳钢、不锈钢、黄铜、铝材
- 实时成本分析和趋势预测
- 供应商价格对比
- 利润保护机制

### 3️⃣ customer_email_dfac_agent.yaml
**基于DFAC流程的客户关系管理师**

**DFAC流程：**
- **Discover** - 邮件发现和初步扫描
- **Filter** - 智能过滤和优先级排序
- **Analyze** - 深度分析和意图识别
- **Categorize** - 分类标记和智能路由

**核心功能：**
- 🌍 多语言支持 (中英西德法等)
- ⚡ 2小时内响应承诺
- ❤️ 客户情感分析和关系维护
- 📊 客户评分和商机优先级

### 4️⃣ project_tracking_agent.yaml
**看板式项目管理专家**

**项目全生命周期：**
```
询价 → 报价 → 订单 → 排期 → 生产 → 检验 → 发货 → 交付 → 售后
```

**核心特色：**
- 📋 看板可视化进度管理
- ⚠️ 延误预测和早期预警
- 📄 文档管理和质量控制
- 💳 实时利润率跟踪

## 🔄 协同工作流程

### 完整业务流程自动化：

1. **客户询价阶段**
   - 邮件智能体自动分类和回复
   - 图纸分析智能体并行技术评估
   - 客户评分和紧急程度识别

2. **报价准备阶段**
   - 成本控制智能体实时价格核算
   - 技术规格和制造方案确认
   - 智能报价策略和利润保护

3. **订单执行阶段**
   - 项目管理智能体看板跟踪
   - 里程碑监控和供应商协调
   - 质量控制和客户沟通

4. **交付完成阶段**
   - 自动化交付确认和满意度调查
   - 财务结算和项目归档
   - 经验总结和知识积累

## 🔧 快速开始

### 1. 环境准备
```bash
# 创建智能体目录
mkdir trading_agents
cd trading_agents

# 安装依赖
pip install paddlepaddle paddleocr
pip install freecad
pip install prophet
pip install transformers
```

### 2. 配置调整
- 修改智能体配置文件中的API密钥和数据库连接
- 根据公司业务流程调整触发器和规则
- 设置客户特定的通知模板和语言偏好

### 3. 系统集成
- 连接到现有CRM/ERP系统
- 配置邮件服务器和供应商API
- 设置通知渠道和权限管理

### 4. 测试验证
- 从邮件智能体开始测试
- 逐步验证每个智能体的功能
- 测试智能体之间的协同工作
