# 紧固件产品分类体系
## 从生产技术角度的分类标准

### 一级分类：定制件 vs 标准件

#### 1. 标准件 (Standard Fasteners)
**定义**: 按照国际标准(ISO)、国家标准(GB)、行业标准生产的通用紧固件
**特点**: 批量生产、规格标准化、互换性强、成本较低

##### 标准件次级分类:

**1.1 螺纹紧固件 (Threaded Fasteners)**
- **螺栓类 (Bolts)**
  - 六角螺栓 Hex Bolts
    - 外六角 bolts (Hex Head Bolts)
    - 内六角 bolts (Socket Head Bolts)
    - 六角法兰螺栓 Hex Flange Bolts
  - 方头螺栓 Square Head Bolts
  - 沉头螺栓 Countersunk Bolts
  - 半圆头螺栓 Round Head Bolts
  - T型螺栓 T-Bolts
  - 地脚螺栓 Foundation Bolts

- **螺钉类 (Screws)**
  - 自攻螺钉 Self-Tapping Screws
    - 自钻自攻 screws
    - 板牙 screws
    - 尾牙 screws
  - 机制螺钉 Machine Screws
    - 盘头 screws
    - 沉头 screws
    - 圆柱头 screws
  - 木螺钉 Wood Screws
  - 干壁螺钉 Drywall Screws
  - 钻尾螺钉 Drilling Screws
    - 钢结构钻尾
    - 铝合金钻尾

- **螺母类 (Nuts)**
  - 六角螺母 Hex Nuts
    - 标准螺母
    - 薄型螺母
    - 加厚螺母
    - 法兰螺母
  - 锁紧螺母 Lock Nuts
    - 尼龙锁紧螺母
    - 金属锁紧螺母
  - 盖形螺母 Cap Nuts
  - 焊接螺母 Weld Nuts
  - 蝶形螺母 Wing Nuts

- **垫圈类 (Washers)**
  - 平垫圈 Flat Washers
  - 弹簧垫圈 Spring Washers
  - 齿形锁紧垫圈 Lock Washers
  - 方垫圈 Square Washers

**1.2 非螺纹紧固件 (Non-Threaded Fasteners)**
- **销类 (Pins)**
  - 圆柱销 Dowel Pins
  - 锥形销 Taper Pins
  - 开口销 Split Pins (Cotter Pins)
  - 弹性圆柱销 Spring Pins
  - 定位销 Positioning Pins

- **铆钉类 (Rivets)**
  - 实心铆钉 Solid Rivets
  - 空心铆钉 Blind Rivets
  - 半空心铆钉 Semi-Tubular Rivets
  - 拉铆钉 Pull Rivets

- **挡圈类 (Retaining Rings)**
  - 孔用挡圈 External Retaining Rings
  - 轴用挡圈 Internal Retaining Rings
  - E型挡圈 E-Clips

#### 2. 定制件 (Custom Fasteners)
**定义**: 根据客户特定要求设计生产，非标准规格的紧固件
**特点**: 专门设计、小批量生产、定制化程度高、价格相对较高

##### 定制件次级分类:

**2.1 按定制程度分类**
- **轻度定制**: 标准件的尺寸或材料微调
- **中度定制**: 结构改进或特殊处理
- **高度定制**: 全新设计，复杂结构

**2.2 按技术特点分类**

**特殊材料定制件**
- 不锈钢定制 (SS304, SS316, SS321等)
- 钛合金定制 (Grade 2, Grade 5等)
- 铝合金定制 (6061, 7075等)
- 合金钢定制 (碳钢、合金钢)
- 塑料定制件 (PEEK, Nylon等)

**特殊表面处理定制**
- 镀层定制 (镀锌、镀铬、镀镍等)
- 涂层定制 (达克罗、特氟龙等)
- 阳极氧化定制 (铝制件)
- 热处理定制 (调质、渗碳等)

**特殊结构定制**
- 异形头部定制
- 特殊螺纹定制
- 复合结构定制
- 组合件定制

**2.3 按应用领域分类**
- **汽车行业定制件**
  - 发动机专用紧固件
  - 底盘专用紧固件
  - 车身专用紧固件

- **电子行业定制件**
  - 电器外壳定制螺丝
  - 精密电子件紧固件
  - 防EMI定制件

- **建筑行业定制件**
  - 幕墙专用紧固件
  - 钢结构专用件
  - 装配式建筑专用件

- **机械设备定制件**
  - 重型设备专用件
  - 精密机械专用件
  - 特殊环境专用件

### 二级分类：按生产标准和图纸

#### 标准件按标准分类
- **国际标准**: ISO 4017 (六角螺栓), ISO 4032 (六角螺母)
- **德国标准**: DIN 933, DIN 931 (六角螺栓)
- **美国标准**: ANSI B18.2.1 (六角螺栓)
- **日本标准**: JIS B1180 (六角螺栓)
- **中国标准**: GB/T 5782 (六角螺栓)

#### 定制件按图纸分类
- **客户图纸**: 按客户提供的技术图纸生产
- **来图加工**: 根据客户图纸进行加工
- **测绘图纸**: 根据实物测绘生产
- **联合设计**: 与客户共同设计生产

### 三级分类：细分类别

#### 材料细分
- **不锈钢类**
  - 304不锈钢 (通用耐腐蚀)
  - 316不锈钢 (高耐腐蚀)
  - 321不锈钢 (耐高温)
  - 双相不锈钢 (高强度)

- **碳钢类**
  - 4.8级 (普通强度)
  - 8.8级 (中高强度)
  - 10.9级 (高强度)
  - 12.9级 (超高强度)

- **合金钢类**
  - 40Cr (调质钢)
  - 35CrMo (高温用钢)
  - 42CrMo (高强度钢)

#### 表面处理细分
- **电镀类**
  - 白锌 (镀锌)
  - 彩锌 (彩色镀锌)
  - 黑锌 (黑镀锌)
  - 镀镍
  - 镀铬
  - 镀铜

- **化学处理类**
  - 达克罗 (Dacromet)
  - 特氟龙 (Teflon)
  - 阳极氧化
  - 磷化处理
  - 发黑处理

#### 精度等级细分
- **普通精度 (C级)**: 一般用途
- **中等精度 (B级)**: 机械装配
- **高精度 (A级)**: 精密机械
- **超高精度 (AA级)**: 航空航天

---

**分类编码建议**:
```
一级分类: 标准(S) / 定制(C)
二级分类: 产品类别 (BOL, SCR, NUT等)
三级分类: 材料代码 (SS304, C45等)
四级分类: 表面处理代码 (ZP, HD等)

示例: S-BOL-SS304-ZP (标准-螺栓-不锈钢304-镀锌)
示例: C-SCR-8.8-HD (定制-螺钉-8.8级-黑锌)
```
