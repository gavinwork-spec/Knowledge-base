### 二级分类：按图纸和技术标准

#### 2.1 按生产图纸分类

**标准图纸产品 (Standard Drawings)**
- **国标产品** - 按照中国国家标准(GB)生产
- **行标产品** - 按照行业标准生产
- **企业标准** - 企业内部标准产品
- **通用设计** - 市场通用设计图纸

**定制图纸产品 (Custom Drawings)**
- **客户图纸** - 完全按照客户图纸生产
- **来图加工** - 客户提供图纸，我们加工
- **测绘仿制** - 根据实物测绘生产
- **联合设计** - 与客户共同设计开发

**OEM/ODM产品 (OEM/ODM Products)**
- **OEM生产** - 按品牌商要求生产
- **ODM开发** - 自主设计，贴牌生产
- **SKD组装** - 半成品组装

#### 2.2 按技术标准分类

**材料标准 (Material Standards)**
- **实木类 Solid Wood**
  - 橡木 Oak
  - 胡桃木 Walnut
  - 樱桃木 Cherry
  - 松木 Pine
  - 橡胶木 Rubberwood
  - 白蜡木 Ash

- **人造板类 Engineered Wood**
  - 中密度板 MDF
  - 高密度板 HDF
  - 刨花板 Particle Board
  - 多层板 Plywood
  - 细木工板 Blockboard
  - 密度板 Density Board

- **金属材料 Metal Materials**
  - 不锈钢 Stainless Steel
  - 铝合金 Aluminum Alloy
  - 钢材 Steel
  - 铁材 Iron

- **塑料材料 Plastic Materials**
  - PP (聚丙烯)
  - PE (聚乙烯)
  * ABS工程塑料
  * 有机玻璃 Acrylic

**结构标准 (Structural Standards)**
- **框架结构 Frame Structure**
  - 实木框架 Solid Wood Frame
  - 板式框架 Panel Frame
  - 金属框架 Metal Frame
  - 混合框架 Hybrid Frame

- **连接方式 Connection Methods**
  - 榫卯结构 Mortise and Tenon
  - 五金连接 Hardware Connection
  - 胶合连接 Adhesive Bonding
  - 焊接连接 Welding
  - 螺纹连接 Threaded Connection

**表面处理标准 (Surface Finish Standards)**
- **油漆涂装 Painting**
  - 水性漆 Water-based Paint
  - 油性漆 Oil-based Paint
  - UV漆 UV Coating
  - 硝基漆 Nitrocellulose Lacquer

- **表面贴面 Surface Veneer**
  - 天然木皮 Natural Veneer
  - 科技木皮 Engineered Veneer
  - 热转印膜 Thermal Transfer Film
  - 三聚氰胺板 Melamine Board

### 三级分类：细分类别

#### 3.1 按生产工艺细分

**实木家具生产工艺**
- **传统工艺 Traditional Craft**
  - 手工雕刻 Hand Carving
  - 榫卯结构 Mortise and Tenon
  - 传统涂装 Traditional Finishing

- **现代工艺 Modern Process**
  - 数控加工 CNC Machining
  - 机械化生产 Mechanized Production
  - 流水线装配 Assembly Line

**板式家具生产工艺**
- **32mm系统 32mm System**
  - 板材切割 Panel Cutting
  - 封边处理 Edge Banding
  - 钻孔加工 Drilling
  - 连接组装 Assembly

- **定制工艺 Custom Process**
  - 尺寸定制 Size Customization
  - 功能定制 Function Customization
  - 外观定制 Appearance Customization

#### 3.2 按功能细分

**功能导向分类**
- **收纳型家具 Storage Furniture**
  - 衣柜 Wardrobes
  - 书柜 Bookcases
  - 储物柜 Storage Cabinets
  - 鞋柜 Shoe Cabinets

- **工作型家具 Work Furniture**
  - 书桌 Desks
  - 工作台 Workbenches
  - 办公桌 Office Desks
  - 实验台 Lab Tables

- **休息型家具 Leisure Furniture**
  - 沙发 Sofas
  - 躺椅 Lounge Chairs
  - 躺床 Daybeds
  - 吊椅 Hammock Chairs

**使用场景细分**
- **小户型家具 Compact Furniture**
  - 多功能家具 Multi-functional
  - 可折叠家具 Foldable
  - 空间节省 Space-saving

- **大户型家具 Large Space Furniture**
  - 组合家具 Modular
  - 豪华家具 Luxury
  - 定制家具 Custom-made

#### 3.3 按质量等级细分

**质量等级分类**
- **经济型 Economy Grade**
  - 材料：人造板为主
  - 工艺：基础工艺
  - 价格：经济实惠

- **标准型 Standard Grade**
  - 材料：实木与板材结合
  - 工艺：标准工艺
  - 价格：中等价位

- **高档型 Premium Grade**
  - 材料：实木为主
  - 工艺：精细工艺
  - 价格：高端价位

- **豪华型 Luxury Grade**
  - 材料：名贵实木
  - 工艺：手工精作
  - 价格：奢华价位

#### 3.4 按环保标准细分

**环保等级分类**
- **E0级** - 甲醛释放量≤0.05mg/L
- **E1级** - 甲醛释放量≤0.12mg/L
- **无醛级** - 零甲醛添加
- **CARB认证** - 加州空气资源委员会认证

**绿色环保标准**
- **FSC认证** - 森林管理委员会认证
- **PEFC认证** - 森林认证体系认可计划
- **绿色建材** - 国家绿色建材认证

**分类编码建议**:
```
一级分类: 家具 (FUR)
二级分类: 大类 (OFC/RES/OUT/HTL/COM/EDU/MED)
三级分类: 细分类 (CHA/SOF/TAB/BED等)
四级分类: 材料代码 (SS/OAK/PIN/MDF等)
五级分类: 等级代码 (ECO/STD/PRE/LUX)

示例: FUR-OFC-CHA-OAK-STD (家具-办公-椅子-橡木-标准)
示例: FUR-RES-BED-MDF-ECO (家具-民用-床-中纤板-经济)
```
