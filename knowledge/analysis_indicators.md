# 分析指标定义文档

## 概述
本文档定义了知识库系统中用于业务分析的关键指标、计算公式和业务含义。

## 价格相关指标

### 1. 平均价格 (Average Price)
**定义**: 在指定时间范围内的产品平均报价

**计算公式**:
```
平均价格 = Σ(报价金额) / 报价次数
```

**业务含义**:
- 反映产品的市场均价水平
- 用于价格趋势分析和对比
- 作为定价策略的参考基准

**维度**: 按产品类别、工厂、时间周期统计

### 2. 价格变动率 (Price Change Rate)
**定义**: 相对于上一周期的价格变化百分比

**计算公式**:
```
价格变动率 = (本期平均价格 - 上期平均价格) / 上期平均价格 × 100%
```

**业务含义**:
- 衡量价格波动程度
- 识别价格趋势（上涨/下跌）
- 预测未来价格走向

**分类**:
- **正常波动**: ±5%以内
- **显著变化**: ±5%~15%
- **大幅波动**: 超过±15%

### 3. 价格区间 (Price Range)
**定义**: 价格的分布范围

**计算公式**:
```
价格区间 = [最低价格, 最高价格]
价格差值 = 最高价格 - 最低价格
```

**业务含义**:
- 反映价格离散程度
- 识别异常报价
- 评估市场竞争力

### 4. 价格标准差 (Price Standard Deviation)
**定义**: 价格波动的统计指标

**计算公式**:
```
价格标准差 = √(Σ(价格 - 平均价格)² / (n-1))
```

**业务含义**:
- 衡量价格稳定性
- 识别价格异常值
- 评估定价一致性

## 业务量指标

### 5. 报价频率 (Quote Frequency)
**定义**: 单位时间内的报价次数

**计算公式**:
```
报价频率 = 报价次数 / 时间周期
月报价频率 = 月报价次数
周报价频率 = 周报价次数
```

**业务含义**:
- 反映业务活跃程度
- 评估客户需求强度
- 预测业务量趋势

### 6. 客户覆盖率 (Customer Coverage Rate)
**定义**: 有报价记录的客户占总客户的比例

**计算公式**:
```
客户覆盖率 = (有报价记录的客户数 / 总客户数) × 100%
```

**业务含义**:
- 衡量市场渗透程度
- 识别未开发客户
- 评估销售覆盖效果

### 7. 产品覆盖率 (Product Coverage Rate)
**定义**: 有报价记录的产品类别占总产品类别的比例

**计算公式**:
```
产品覆盖率 = (有报价记录的产品类别数 / 总产品类别数) × 100%
```

**业务含义**:
- 反映产品线完整性
- 识别产品空白区域
- 指导产品开发策略

## 工厂表现指标

### 8. 工厂活跃度 (Factory Activity Level)
**定义**: 工厂在指定时间内的业务活跃程度

**计算公式**:
```
工厂活跃度 = 报价次数 × 平均报价金额 × 客户数量权重
客户数量权重 = 客户数量 / 最大客户数量
```

**业务含义**:
- 评估工厂综合表现
- 识别核心供应商
- 优化供应商组合

### 9. 工厂响应率 (Factory Response Rate)
**定义**: 工厂对询价的响应比例

**计算公式**:
```
工厂响应率 = (报价次数 / 询价次数) × 100%
```

**业务含义**:
- 衡量服务质量和配合度
- 评估供应商可靠性
- 优化供应商选择

### 10. 价格竞争力指数 (Price Competitiveness Index)
**定义**: 工厂价格相对于市场平均水平的竞争力

**计算公式**:
```
价格竞争力指数 = (市场平均价格 - 工厂平均价格) / 市场平均价格 × 100%
正值表示价格优势，负值表示价格劣势
```

**业务含义**:
- 评估价格竞争力
- 指导采购决策
- 优化成本控制

## 质量指标

### 11. 分类准确率 (Classification Accuracy Rate)
**定义**: 自动分类结果的准确程度

**计算公式**:
```
分类准确率 = (正确分类的记录数 / 总分类记录数) × 100%
```

**业务含义**:
- 衡量分类算法效果
- 指导算法优化
- 评估数据质量

### 12. 数据完整率 (Data Completeness Rate)
**定义**: 必填字段完整的数据记录比例

**计算公式**:
```
数据完整率 = (关键字段完整的记录数 / 总记录数) × 100%
关键字段：客户名称、联系方式、产品名称等
```

**业务含义**:
- 评估数据质量
- 指导数据清理
- 确保分析准确性

## 时间分析指标

### 13. 业务周期 (Business Cycle)
**定义**: 从询价到成交的平均时间

**计算公式**:
```
业务周期 = Σ(成交日期 - 询价日期) / 成交订单数
```

**业务含义**:
- 评估销售效率
- 优化业务流程
- 改善客户体验

### 14. 季节性指数 (Seasonality Index)
**定义**: 业务量的季节性波动指标

**计算公式**:
```
季节性指数 = (当期业务量 / 平均业务量) × 100%
平均业务量 = 年总业务量 / 12
```

**业务含义**:
- 识别业务季节性规律
- 制定库存策略
- 优化资源配置

## 趋势分析指标

### 15. 增长率 (Growth Rate)
**定义**: 业务量相对于上一期的增长百分比

**计算公式**:
```
增长率 = (本期数值 - 上期数值) / 上期数值 × 100%
```

**业务含义**:
- 评估业务发展趋势
- 制定增长目标
- 预测未来表现

### 16. 移动平均 (Moving Average)
**定义**: 消除短期波动的趋势指标

**计算公式**:
```
n期移动平均 = (最近n期数值之和) / n
```

**业务含义**:
- 平滑短期波动
- 识别长期趋势
- 预测未来走向

## 异常检测指标

### 17. 异常价格检测 (Price Anomaly Detection)
**定义**: 识别超出正常范围的价格

**检测方法**:
- **IQR方法**: 超出 Q1-1.5×IQR 或 Q3+1.5×IQR
- **Z-score方法**: |Z-score| > 2
- **业务规则**: 偏离平均价格超过±30%

**业务含义**:
- 发现数据质量问题
- 识别特殊交易情况
- 预防业务风险

### 18. 异常频率检测 (Frequency Anomaly Detection)
**定义**: 识别异常高或低的业务频率

**检测方法**:
- **标准差法**: 偏离平均值超过2个标准差
- **百分位法**: 超出95%或低于5%分位数
- **同比分析**: 与去年同期比较超过±50%

## 计算规则和标准

### 数据处理规则
1. **空值处理**: 排除空值参与计算
2. **异常值处理**: 识别但不过滤，单独标记
3. **时间范围**: 按自然月、季度、年度统计
4. **货币单位**: 统一使用人民币（CNY）
5. **精度要求**: 价格保留2位小数，百分比保留1位小数

### 统计周期
- **日报**: 当日数据统计
- **周报**: 周一至周日统计
- **月报**: 自然月统计
- **季报**: 自然季度统计
- **年报**: 自然年度统计

### 对比基准
- **环比**: 与上一周期对比
- **同比**: 与去年同期对比
- **基期**: 与指定基准期对比

## 指标应用场景

### 1. 管理层决策
- **价格趋势分析**: 使用平均价格、价格变动率
- **供应商评估**: 使用工厂活跃度、价格竞争力指数
- **业务规划**: 使用增长率、季节性指数

### 2. 销售团队应用
- **客户管理**: 使用客户覆盖率、报价频率
- **产品策略**: 使用产品覆盖率、价格区间
- **绩效评估**: 使用业务周期、响应率

### 3. 采购部门应用
- **供应商选择**: 使用价格竞争力指数、工厂响应率
- **成本控制**: 使用价格变动率、价格标准差
- **风险管理**: 使用异常价格检测

### 4. 数据质量监控
- **分类效果**: 使用分类准确率
- **数据完整性**: 使用数据完整率
- **异常监控**: 使用异常检测指标

## 指标阈值标准

### 正常范围定义
- **增长率**: -10% ~ +30% 为正常范围
- **价格变动率**: -5% ~ +5% 为稳定
- **响应率**: >80% 为良好
- **分类准确率**: >90% 为优秀
- **数据完整率**: >95% 为优秀

### 预警级别
- **绿色预警**: 指标在正常范围内
- **黄色预警**: 指标偏离正常范围10-20%
- **红色预警**: 指标偏离正常范围超过20%
